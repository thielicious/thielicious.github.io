aLocal

Overview



aLocal ('a': abbr. for 'auto') is a tiny Javascript snippet which automatically updates the address bar into localhost.

It's useful for Beginners who just started to learn server-side languages like PHP. Apart from many other solutions in other languages, this one simply executes it after opening a file into the browser. It is implemented just like any other script at the top.
