Beginners who are not fond in such things start with XAMPP most likely. There are many other ways to fix it in almost any language or directly in host files on Windows.

This is an example for a solution in plain JavaScript:

aLocal ('a' for auto) is a tiny JavaScript snippet which automatically updates the address bar to localhost - ready for server-side scripts. An alternativity for newbies who plan to start programming in PHP.


SETUP INFORMATION



aLocal.js requires no extern libraries or anything similar.

  • 1. Follow that link below and choose your download option.

    - aLocal.js

  • 2. Place this script in your root directory of your host.
  • 3. Open to edit a PHP file which must be inside of the root path.
  • 4. Embed alocal.js in your page header just like any other linked script:
    • 		
	<script src="[filepath]/aLocal.js"></script>
  • 5. Save it and done.


    • CONFIGURATION



    The configuration is set directly in the script file.

    • .setPath(path)

      defines the root path, default: XAMPP folder

    • .setLocal(server)

      defines your server, default: localhost

    • .change()

      executes the change




    Examples:


    This code below is already set by default. There is no extra configuration needed.
    		
	var alocal = new aLocal();
	alocal.change();
    The following example shows you how to configure the user-defined way.
    		
	var alocal = new aLocal();
	alocal.setPath('file:///C:/path/to/your/root/folder/'); <-- optional
	alocal.setLocal('http://localhost or user-defined domain/'); <-- optional
	alocal.change();


    USAGE



    Once embedded, it's ready to perform. Simply, open a PHP file in your browser, no need to alter the annoying address bar anymore. It is not a special thing but a timesaver.
