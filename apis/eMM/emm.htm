eMM

Overview



eMM (abbr. for 'eMail Manager') is a compact and tiny form mailer including all its adjustments for personal desires. This API simplifies the flexible use of several input fields which will be used to send and transport data through a classical mailing process. Useful for setting up customer contact forms and newsletters.
